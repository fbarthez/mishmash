Animated PNG examples

1) APNG, with WebKit transforms

This APNG file animates in Firefox 3+ and Opera 9.5+. For WebKit browsers (e.g. Safari and Chrome), the APNG shows its first, static frame, to which a CSS rotation animation is applied. No JavaScript is used.

2) JavaScript spinner

This could be used for browsers that don't support APNG or CSS animations (e.g. Internet Explorer). A sprite image that contains each of the frames is used. The sprite is applied as the background image for an element, and the background position is progressively changed.

In this example, the sprite image is included within the JavaScript as a dataURL, which isn't supported by IE6 or IE7 (but is supported by IE8+). This is just for convenience - the image could be included as an external file, which would allow support in IE6 and IE7 too.
